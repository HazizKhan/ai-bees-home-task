## Setup

1. install nestJs - `npm i -g @nestjs/cli`
2. Install Docker
3. convert `.env.local` to `.env`
3. Start development server `sudo docker-compose up dev`
4. If you want to build again then try this command `sudo docker-compose up --build dev`
